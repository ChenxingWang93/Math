- f(x): Start with some system to study //系统
- (dx): Change by the smallest amount possible //以最小的变化量
- f(x + dx) - f(x): 前 &后变化关系
- f(x + dx) - f(x) /dx: We don't care how small (dx) is: get the rate of motion through the continuum //无论(dx)的大小，得到穿越连续统的变化率
